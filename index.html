Declarations

Let

Let has block scope if it is declared in a block and global scope if it is declared outside of a block. Also, it not accessible before it is declared. You can not redeclare the same variable in the same scope. Ex. Let me = ‘Foo’;
Let me = ‘Bar’; This would make an error.

Const

Primitives can not be changed. If you make an object with the const keyword its properties and method can still be added, subtracted, or changed. BUT you can not totally reassign the object to be a new object.

Var

Var has function scope if declared in a function and global scope if declared outside of a function.

Variables declared Globally (outside any function) have Global Scope. They can be used anywhere in the program.

Operators

Unary Operators

Some operator like typeof can be use with one operator. The minus symbol can be used both as a Unary or Binary operator.

Comparisons

3>2 = true; Only one value does not equal itself Nan. So NaN == NaN is false. NOTE: null == null and undefined == undefined. This makes sense with numbers but what about words? Well letters later in the alphabet have higher values. So a is less than b. But! Capitals have lower values. So a capital B is less than a lowercase a. Special characters are even lower.

LOW - empty string - Punctuation - Capitals - Lowercase Letters - Later letters - HIGH

Logical Operators

&&- and (true && false) = false
||- or (true || false) = false
!- not Flips value. !true is false. !false is true.

Ternary Operator

Condition ? 1 : 2. If the condition is true use the first value if not use the second.

Short-Circuitng of Logical Operators

The || operator, for example, will return the value to its left when that can be converted to true and will return the value on its right otherwise.  Console.log(null || “Hello”); This would log Hello because null is a false value.

The && operator works similarly but the other way around. When the value to its left is something that converts to false, it returns that value, and otherwise it returns the value on its right.
“"&&"Hello" would return “” and “Something”&&"Hello" would return Hello.

In the case of true || X, no matter what Xis—even if it’s a piece of program that does something terrible—the result will be true, and X is never evaluated. The same goes for false && X, which is false and will ignore X. This is called short-circuit evaluation.

Functions

Creating Functions

Functions have the function keyword, parameters, and then a code block.

A function is created with parameters and when it is called arguments are passed into the function.

statements;
}

Once you make a function to use it you need to call it.

This is how you call a function. myX and myY are arguments that are passed into the function. myX would be used inplace of x.

Bindings

Bindings hold values. They use a keyword, a name, and a value. Bindings can change value by using the assignment operator. The value of a binding without a value is undefined. Two diffrent bindings can have the same value.

Prompt

To set a value as being equal to user input you need to use prompt().

prompt(text, defaultText)

It takes two parameters that are both strings. The first is the question that the user answers. The second is optional and it containes placeholder text. It returns whatever the user inputs. If they input nothing you get an empty string and if they hit cancel you get null.

Array Methods

Adding and Deleting Items

  • pop()

    • Removes the last item and returns its value.

    No parameters.

  • shift()

    • Removes the first item and returns the removed item.

    No parameters.

  • push()

    • Adds an item to the end and retuns the new array lenght.

    The parameters are the items you want to add.

  • unshift()

    • Adds an item to the start and returns the lenght of the array.

  • concat()

    • This can combine two or more arrays. The arrays that you want to add should be passed as arguments.

    arr1.concat(arr2, arr3);

    This example would combine 3 arrays.

  • splice()

    • Splice has may parameters. The first is where the new elements should be added and the second is how many elements should be deleted.

    For example 2 , 0 would place the elements after the first two and delete no elements.

    2 , 1 would place them after the first two elements and then delete the third.

    Every paramater after is an element that will be added.

    let heros = ["Superman", "Wonderwoman", "Aquaman"];
    heros.splice(1,1, "Batman");

    This code will skip the first element which is Superman. Then it will delete one element being Wonder Woman. Then it will add Batman after Superman. The result would be an array with Superman, Batman, and Aquaman.

  • slice()

    • Slice will create a new array with parts of an existing array. It does not impact the inital array.

    Note that the parameters passed into slice represent the keys to the array and not the number of their position. This means that the first element is 0 and the second is 1.

    let arr1 = [1,2,3];
    let arr2 = arr1.slice( 0, 1);

    In this example arr1 does not change but arr2 had 1 and 2 in it.

Date

Making Dates

Date is an object. You can create a date with the Date constructor. It has 7 parameters: year, month, day, hour, minute, second, and millisecond. Dates are stored in milliseconds.

let today = new Date();
let d = new Date(2018, 12, 14);

Date Methods

To display a date you need to change it into a string with the toString() method. Or you can use toDateString() which prints is in an easier to read format.

console.log(today.toString());

Get Date Methods

There are many ways to get part of a date like getFullYear, getMonth, getDay... These will get you one part of the date. Note that they count from 0. so for getHour you will get a number from 0-23 and for getMonth you will get a number from 0-11.

Set Date Methods

Set Date methods work alot like get Date methods. They set the value of part of a date. The names are the same but they use set instead of get. They still count from 0.

var today = new Date();
today.setFullYear(2090);

In this example today is set to the current date and then in the next line the year is changed to 2090.

Events

Events

Events start with the word on. Some common events are onclick, onmouseover,

Event Listeners

Event Listeners allow you to add multiple functions to one event. These functions are added natively. They do not change the value of the event. This makes it faster because the code it read with the rest of the native JavaScript. You can not remove the functions once you made them because they are anonymous. The way to get around this is to make the function outside of the event.

Here is a list of Events!

How to make an event listeners!

  1. Target the object. You can do this with document.getElementById('name');
  2. Afterward write .addEventListener();.
  3. Then add an event without the on part in the (). So onclick would be just click. So it would be .addEventListener("click");
  4. Make the next the parameter the function. Now it would be ("click", function(){console.log("Hello World!");});
  5. To make another event you repeat the eariler steps.
document.getElementById('name').addEventListener("click", function(){console.log("Hello World!");});
document.getElementById('name').addEventListener("click", function(){console.log("It's me again!");});

Removing an event!

If you made an event outside of the function and you added it then it can be removed!

document.getElementById('name').addEventListener("click", helloWorld); //helloWorld is a function outside of the event.
document.getElementById('name').removeEventListener("click", helloWorld); //This can delete the function because it has a name

HTML Events

HTML events are not used much anymore because it is better to have your Javascript all together.

DOM On-Event Handlers

These are better than HTML events but you can only use one function so people tend to prefer event listeners. These events are attached to the object. The reason you can only have one function with this method is because you are changing the object property to a method and there is only one function allowed in a method. In the example below you are changing the onclick property to a method with the function that you are sett ing it equal to.

How to make an Event Handler!

  1. Select an element. You can do this with document.getElementById('name');
  2. Type the name of the event that will trigger the function.
  3. Set the element.event equal to the function you want triggerd. NOTE: Do not call the function.
document.getElementById('name').onclick = function(){console.log("Hello World!");};
